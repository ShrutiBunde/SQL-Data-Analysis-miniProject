Employee Management System - 

This is a simple Employee Management System created using Flask, MySQL, HTML, and CSS.

It allows users to:

Add employee details

View all employee records stored in the database

Technologies Used - 

Python (Flask)
MySQL
HTML
CSS

Project Structure - 

project/

├── app.py

├── templates/

│   └── index.html

├── static/

│   └── style.css

└── README.md

How to Run the Project - 

Install required packages:

pip install flask mysql-connector-python

Update MySQL credentials in app.py.

Run the application:

python app.py

Open browser and go to:

http://127.0.0.1:5000/

Functionality

User enters employee details in form

Data is saved in MySQL database

All records are displayed in a table

Uses Flask with Jinja2 templating


Note -

This is a basic Flask + MySQL project meant for learning purposes.
